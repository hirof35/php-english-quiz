# 記述式・英単語クイズゲーム (SQLite連携版)

PHPと組み込み型データベース（SQLite3）を利用した、Webブラウザ上で動作するシンプルな英単語記述式クイズアプリケーションです。
面倒な外部データベースサーバー（MySQL等）の設定を必要とせず、単一のPHPファイルを配置するだけで即座に動作します。

## 🚀 特徴

- **SQLite3自動セットアップ**: 初回アクセス時に自動的にデータベースファイル（`quiz.db`）および初期サンプルデータが生成されます。
- **自由入力（記述式）判定**: 選択肢形式ではなく、キーボードからスペルを直接入力する本格的なクイズ形式です。
- **入力揺れの吸収**: 前後の余分なスペースの自動除去（`trim`）および、大文字・小文字を区別しない判定アルゴリズムにより、直感的な回答判定を行います。
- **状態管理**: `$_SESSION` を用いて、進行状況やスコアをセッション内で安全に管理します。

## 🛠️ 動作環境

- PHP 7.4 / 8.x 以上
- PHP PDO 拡張モジュール（SQLite3 ドライバ有効化）
- Webサーバー（Apache, Nginx, または PHP組み込み開発サーバー）

## 📦 設置・起動手順

1. **ファイルの配置**
   提供されたソースコードを `quiz_db.php` などのファイル名で保存し、PHPが動作するWebサーバーのドキュメントルートに配置します。

2. **ディレクトリ権限の確認**
   SQLiteは同一ディレクトリ内に `quiz.db` ファイルを自動生成するため、対象ディレクトリにWebサーバー（`www-data` や `_www` など）の**書き込み権限**があることを確認してください。

3. **PHP組み込みサーバーでのクイック起動**
   ローカル環境で手軽にテストする場合、ターミナルでファイルのあるディレクトリに移動し、以下のコマンドを実行します。
